# lita-y

Y? Pls

## Installation

Add lita-y to your Lita instance's Gemfile:

``` ruby
gem "lita-y"
```

## Usage

```
Dolan: Bogs is raep tiem, prepar anis
Bogs: Dolan y?
Lita: Bogs pls
```
